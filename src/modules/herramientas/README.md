# Módulo Herramientas de Creación con IA

Módulo 3 del documento técnico — motor de valor. 6 herramientas: planeaciones, rúbricas, banco de prompts, adaptador de contenido por nivel, evaluaciones/quizzes, comunicación con familias.

**Fase 1 (MVP):** Planeaciones, Rúbricas y Banco de Prompts.
**Fase 2:** las 6 completas + exportación DOCX/PDF/Google Docs.
Persistir historial de generaciones por usuario (límite freemium: 5/mes).
