# Módulo TMAID (Test de Madurez IA Docente)

Módulo 1 del documento técnico — **core diferenciador**. 10-12 preguntas (4 dimensiones: Conocimiento IA, Uso de herramientas, Integración al aula, Actitud ante el cambio) → nivel asignado + perfil pedagógico-IA + plan de ruta personalizada (Explorar → Aplicar → Dominar).

**Fase 2:** flujo real preguntas → scoring → perfil, con generación de reporte vía prompt estructurado a GPT-4o. Resultado persiste en `resultado_tmaid`. Re-diagnóstico cada 90 días.
