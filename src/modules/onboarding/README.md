# Módulo Onboarding

Módulo 0 del documento técnico: formulario de contexto inicial (5-7 preguntas: nivel educativo, materia, país, uso previo de IA, mayor desafío) que genera el Perfil Docente Base (PDB).

**Fase 1:** formulario con lógica condicional + persistencia del PDB en Supabase (tabla `perfil_docente`).
